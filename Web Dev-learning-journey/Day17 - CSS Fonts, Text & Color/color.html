Let's learn about colors

Colors in CSS can be represented in various ways:

  1. Color Keywords (e.g., red, blue, green)
  2. Hex Color Code (e.g., #FF5733)
  3. RGB (e.g., rgb(255, 0, 0))
  4. RGBA (e.g., rgba(255, 0, 0, 0.5) - with transparency)
  5. HSL (e.g., hsl(120, 100%, 50%) - Hue, Saturation, Lightness)
